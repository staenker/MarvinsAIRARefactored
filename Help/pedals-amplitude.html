Pedals

Amplitude

Define global minimum/maximum motor amplitude and shaping curve used by all pedal effects.

Minimum

Lowest amplitude used at effect onset.

Default: 10%

Range: 0 → 100%

Maximum

Highest amplitude at full intensity.

Default: 100%

Range: 0 → 100%

Curve

Shapes how quickly amplitude rises from Minimum to Maximum.

Default: 0% (linear)

Range: -100 → 100%
