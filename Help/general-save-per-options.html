Save Per Wheel/Car/Track/Etc Options

What it does

Choose where a setting “lives.” Most setting labels in MAIRA can be right‑clicked to decide whether that value should be saved globally or per context like wheelbase, car, track, track configuration, or wet/dry conditions.

Open the save‑scope menu

  1. Right‑click the label of the setting (not the value).
  2. The Save This Setting window opens.

Choose the scope

  • Per Wheelbase: different values for each connected base.
  • Per Car: separate values per car.
  • Per Track: separate values per track.
  • Per Track Configuration: distinct values for layouts.
  • Per Wet/Dry: values can differ when it’s wet vs. dry.

Click Set (↓) to apply, or Cancel (x) to keep the current scope.

You can combine scopes. For example, a setting can be saved per Car and per Track.

How it works behind the scenes

MAIRA stores a single value per unique context based on the scopes you select. When the active context changes (e.g., you switch cars), MAIRA loads the matching value automatically.
