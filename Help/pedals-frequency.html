Pedals

Frequency

Set the global vibration frequency range and response curve for all pedal effects.

Minimum

The frequency to vibrate at when an effect is at 0% frequency.

Default: 15 Hz

Range: 0 → 50 Hz.

Maximum

The frequency to vibrate at when an effect is at 100% frequency.

Default: 50 Hz

Range: 0 → 50 Hz.

Curve

Shapes how quickly frequency rises from Minimum to Maximum as an effect strengthens.

Negative values emphasize early frequency increase; positive values delay it until higher intensity.

Default: 25%

Range: −100 → 100%
