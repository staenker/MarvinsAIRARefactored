Input Mapping

What it does

Bind physical controls on your wheel, button box, keyboard, or Stream Deck to actions in MAIRA. Each mapping can use an optional Hold (a modifier you keep pressed) and a required Click (the actual trigger).

Open the mapping window

  1. Go to the page that contains the control you want to map.
  2. Right-click the mappable control to open the Update Button Mappings window.

Create or update a mapping

  1. Click the record button to tell MAIRA to start monitoring all the buttons and knobs connected to the computer.
  2. Press the button you want to use as the Hold (optional). Keep it held.
  3. While still holding (or if you don’t need a hold), press the button you want to use as the Click.
  4. Release the buttons. Recording stops automatically after the release.
  5. Click Set (↓) to save the mapping, or Clear (x) to discard.
  6. You can also click Add (+) to create multiple button mappings for the same control.
Tip: If you only press one button during recording, it becomes the Click. If you press two in sequence, the first becomes Hold and the second becomes Click.
