Steering Effects

Calibration

Use MAIRA’s built‑in vJoy integration to map iRacing inputs and run the automated calibration process on the Centripetal Circuit test track.

Part 1 — Map controls in iRacing (using vJoy)

These buttons control a virtual joystick so MAIRA can drive your car in iRacing without you touching the real wheel or pedals.

Steering Wheel Left, Steering Wheel Center, Steering Wheel Right, Steering Wheel 90° Left — simulate steering positions for assignment/calibration.

Minimum Throttle / Maximum Throttle — set throttle axis min/max.

Minimum Brake / Maximum Brake — set brake axis min/max.

Shift Up / Shift Down — set shifter buttons.

  1. Enter a Test session at Centripetal Circuit and open Settings → Driving → Input Calibration in iRacing.
  2. Click on one of the calibrate buttons (e.g., Steering, Pedals, Gearbox)
  3. Follow the iRacing instructions, pressing the matching MAIRA buttons to send the vJoy input. Repeat until all three parts are calibrated.
MAIRA initializes vJoy automatically on this track. If the overlay shows the Centripetal warning, the buttons are disabled until you switch to the correct track.

Part 2 — Run the automated calibration

Leave the garage and put your car on the track.

Run Calibration — starts the calibration routine. MAIRA target‑drives at low speed (≈ 15 km/h) and sweeps steering across the supported range (±450°), sampling yaw‑rate vs. steering angle.

Stop Calibration — cancels the routine. Use this if you need to abort early.

During the run you’ll see a progress indicator and a live calibration graph building in real time. When finished, MAIRA saves a calibration file for the current car/setup; it appears in the Calibration File dropdown on this page.

Pro tip: Don’t touch the wheel or pedal buttons while calibration is running. Give the car room to drive the circle; if a collision occurs, press Stop Calibration, reset the car, and try again.

Troubleshooting

No buttons visible? You’re likely not on Centripetal Circuit. Return to the test track and reopen this page.

Inputs don’t register in iRacing? Make sure iRacing’s control assignment is focused (the row is highlighted) before pressing a MAIRA button; also verify vJoy isn’t claimed by other software.
