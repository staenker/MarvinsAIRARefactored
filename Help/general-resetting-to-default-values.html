Resetting to Default Values

Reset a single setting

  1. Move the mouse over the setting’s value.
  2. Hold the right mouse button for about one second.
  3. The value resets to its default.

Reset everything to defaults

  1. Close MAIRA.
  2. Open your Documents folder.
  3. Go to Documents\MarvinsAIRA Refactored.
  4. Delete or rename Settings.xml.
  5. Launch MAIRA again. A fresh default Settings.xml will be created.

If you’d like to keep a backup of your current configuration, rename the file (for example, Settings.backup.xml) instead of deleting it.

Notes

  • Context‑scoped values (Per Wheel/Car/Track/etc.) are stored inside Settings.xml; removing the file clears those too.
  • You can revert to the old values later by restoring your backed‑up file.
