Steering Effects — Calibration File

Select a calibration file for your current car to enable Steering Effects and view the calibration graph that those effects rely on.

UI Location
Steering Effects page • section Calibration File
What this section does
Lets you choose which calibration file to use for the current car. The selected file defines how MAIRA maps steering angle to expected yaw‑rate; that mapping powers Understeer and Oversteer detection and their related effects.

Calibration file dropdown

  • How it’s populated: MAIRA lists CSV files found in your Calibration folder that match the current car. Only files for the car you’re driving are shown.
  • Disable all Steering Effects: Choose [Steering Effects Disabled] (the first option). This clears the selection and MAIRA treats Steering Effects as disabled.
  • Behavior on change: When you pick a file, MAIRA immediately loads it and updates the graph and any dependent settings.

Where files are stored

Calibration CSVs are saved under your Calibration folder. A typical filename is <Car Screen Name> - <Car Setup Name>.csv. The CSV header is Steering Wheel Angle,Yaw Rate.

About the calibration graph

  • Axes: X‑axis = Steering Wheel Angle (degrees). Y‑axis = Yaw Rate (degrees/second).
  • Grid & labels: The graph draws evenly‑spaced vertical and horizontal grid lines with tick labels so you can read angles and yaw‑rate values at a glance.
  • Data trace: Your recorded calibration data is rendered as a continuous blue line across steering angles, derived from the CSV values.
  • Interplay with thresholds: Understeer and Oversteer thresholds are displayed as yellow (minimum) and red (maximum) lines. As you adjust thresholds, you can see these lines moving on the graph.
