Power

Toggle force feedback output on/off.

Racing Wheel FFB Output Quick Toggle
UI Location
Racing Wheel page • Button labeled Power
Default
On (likely) (inferred; confirm in Settings.cs)
What it does
When On, MAIRA sends FFB to the selected Steering Device. When Off, MAIRA stops sending FFB output.

How it works

  • Press the Power button to toggle FFB output.
  • The setting is saved as a simple on/off value and restored on the next launch.
  • If a button mapping is configured, you can toggle Power from your wheel (hold/click behaviors supported).

Related controls

  • Button mappings — Bind physical buttons to toggle Power. (Default: none)
  • Context switches — Optionally store Power per wheelbase / car / track / layout / wet-dry context. (Default: off)
Map a convenient wheel button (e.g., a thumb button) to quickly kill FFB in the pits or if the car is stuck.

Tips & troubleshooting

  • If Power won’t enable, confirm the correct Steering Device is selected and powered.
  • Hot-plugging hardware? Use the app’s refresh/reinitialize action to re-scan devices and re-bind FFB.
