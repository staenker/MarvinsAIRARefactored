Context Switches
The first three switches below include a Context Switches menu so you can enable/disable them per wheelbase/car/track profile. Others are global.

Center wheel while racing

  • Default: Off
  • What it does: Applies your configured centering torque while racing. Cross‑fades with Center wheel while parked as speed moves through 0→5 mph.
  • Context Switches: Supported — useful to keep centering off for open‑wheelers but on for GT cars.

Center wheel while parked

  • Default: On
  • What it does: Keeps a gentle centering torque while parked (0–5 mph) to prevent wobble. Uses the same centering strength you set in Other Features → Wheel Centering Strength.
  • Context Switches: Supported — toggle per profile as needed.

Fade force feedback when entering or exiting car

  • Default: On
  • What it does: Smoothly fades force‑feedback when entering/exiting the car, changing sessions, or when MAIRA suspends/unsuspends FFB.
  • Timing: Fade‑in ≈ 2.0 s, Fade‑out ≈ 0.5 s.

Enable Logitech RPM lights

  • Default: On
  • What it does: Drives the LED RPM bar on supported Logitech wheels via the Logitech SDK. Lights scale with engine RPM reported by iRacing.
  • Requirements: A compatible Logitech wheel and the Logitech driver/runtime.

Allow up to 200% strength (potentially dangerous)

  • Default: Off
  • What it does: Raises the upper limit for Strength from 100% to 200%.
  • Safety warning: High strengths can be dangerous. Use with care and keep an emergency Power button mapped.

Enable FFB even if iRacing simulator FFB is on

  • Default: Off
  • What it does: Allows MAIRA’s FFB to run even when iRacing’s own FFB is enabled. If this is Off, MAIRA will pause its output whenever iRacing’s Enable Force Feedback is on (to avoid double FFB).
  • Recommendation: Leave this Off unless you know you want both systems active (risk of doubled forces).

Enable simple mode

  • Default: Off
  • What it does: Hides advanced controls in the UI to keep things clean.
