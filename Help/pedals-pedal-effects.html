Clutch, Brake, and Throttle Pedal Effects

Assign and prioritize different effects to pedals.

Pedal Effects

Assign one or more effects to each pedal, with higher priority effects overriding lower priority effects.

  • Gear Change
  • ABS Engaged
  • RPM
  • Understeer Effect
  • Oversteer Effect
  • Seat-of-Pants Effect
  • Wheel Lock
  • Wheel Spin
  • Clutch Slip

Strength

How strongly to play this effect on this pedal.

Default: 100%

Range: 0 → 100%

Test

Press the vibrate button to test the selected effect.

Only one effect can be active on each pedal at a time. The order in which effects are assigned determines the priority of each effect.
