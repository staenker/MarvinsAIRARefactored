Racing Wheel page • section Preview
What this section does
Lets you load a pre‑recorded FFB trace, then overlays a graph of the native 360 Hz vs the processed output. Tweak settings and instantly see the effect.

How to use

  1. Load — Select a pre-recorded FFB file from the list of options.
  2. Tune — Adjust Overall Strength, Algorithm and Output settings; the graph updates in real time so you can compare before/after.

What you’ll see

  • Native 360 Hz — the original FFB signal from the recording, in red color.
  • Output torque — the signal after Overall Strength → Algorithm → Curve/Min/Max → Soft Clipping, in blue color.

Tips

  • Use a recording with curbs and braking zones to test both high‑frequency detail and large peaks.
  • Try toggling between algorithms to understand how each shapes the same input.
