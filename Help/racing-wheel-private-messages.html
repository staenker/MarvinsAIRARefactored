Racing Wheel

Private Messages

Control which notifications MAIRA sends to you via iRacing’s chat — as private messages to your driver only.

Send private chat messages to the driver

Enables all MAIRA private messages. If this is Off, no chat messages are sent, regardless of the other toggles.

Default: On

Enable input mapped setting update messages

When you adjust settings via mapped buttons/encoders (e.g., Strength, Max Force, Auto Margin, Curve, etc.), MAIRA sends a short confirmation message showing the new value.

Default: On

Examples

(MAIRA) Strength 75%
(MAIRA) Max Force 12.0 Nm
(MAIRA) Curve 10%

Enable crash protection messages

Sends a one‑time “Crash Protection activated” PM when a high‑g event triggers protection. Messages are rate‑limited: they only send when protection transitions from inactive → active.

Default: Off

Enable curb protection messages

Sends “Curb Protection activated” when a curb/shock event triggers protection (also sent only on the transition into active).

Default: Off

Pro tip: Keep Enable input mapped setting update messages on while dialing in your setup so you get quick confirmation of changes. Turn the feature off for races if you prefer a clean chat window.
