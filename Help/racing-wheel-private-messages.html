Racing Wheel — Private Messages

Control which notifications MAIRA sends to you via iRacing’s chat — as private messages to your driver only.

Racing Wheel Private Messages Notifications
UI Location
Racing Wheel page • section Private Messages
How messages are sent
MAIRA sends a private chat line to you using iRacing’s “whisper” format. Messages sent to the chat window looks like /{playerName} (MAIRA) <message>. The master switch must be On, and you must be in‑session (MAIRA reads your driver name from the sim).

Send private chat messages to the driver

  • Default: On
  • What it does: Enables all MAIRA private messages. If this is Off, no chat messages are sent, regardless of the other toggles.

Enable input mapped setting update messages

  • Default: On
  • What it does: When you adjust settings via mapped buttons/encoders (e.g., Strength, Max Force, Auto Margin, Curve, etc.), MAIRA sends a short confirmation message showing the new value.
  • Examples: “(MAIRA) Strength 75%”, “(MAIRA) Max Force 12.0 Nm”, “(MAIRA) Curve 10%”.

Enable crash protection messages

  • Default: Off
  • What it does: Sends a one‑time “Crash Protection activated” PM when a high‑g event triggers protection. Messages are rate‑limited: they only send when protection transitions from inactive → active.

Enable curb protection messages

  • Default: Off
  • What it does: Sends “Curb Protection activated” when a curb/shock event triggers protection (also sent only on the transition into active).
Pro tip: Keep Enable input mapped setting update messages on while dialing in your setup so you get quick confirmation of changes. Turn the feature off for races if you prefer a clean chat window.
