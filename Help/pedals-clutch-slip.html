Pedals

Clutch Slip

Cue when the clutch pedal is pressed between the minimum and maximum amount.

Starting Point

How much you would need to press the clutch pedal before you start to feel this effect.

Default: 25%

Range: 0 → 100%

Ending Point

How much you would need to press the clutch pedal before you stop feeling this effect.

Default: 75%

Range: 0 → 100%

Frequency

The frequency to use for this effect, as a percentage between the global minimum and maximum frequency.

Default: 100%

Range: 0 → 100%
